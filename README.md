# Ivanti (ivanti)
Ivanti is an IT asset management and security platform providing unified endpoint management, patch management, and IT service management. The Ivanti Neurons product family exposes REST APIs across People & Devices, MDM, ITSM, and Zero-Trust Access, alongside Endpoint Manager APIs for patch and software distribution.
